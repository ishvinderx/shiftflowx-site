# Deployment

This website auto-deploys to Vercel when you push to the `main` branch.

## Setup (one-time)
1. Open GitHub Desktop → Add Existing Repository → select this folder
2. Publish to GitHub (private or public)
3. Go to vercel.com → New Project → Import from GitHub → select this repo
4. Vercel will auto-detect Next.js — click Deploy
5. Every future push from GitHub Desktop will auto-deploy

## Push changes
GitHub Desktop → Commit → Push → Vercel auto-deploys in ~60 seconds
